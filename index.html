02

SKILL

제가 할 수 있는 기술을 소개합니다.

HTML5

웹 표준과 웹 접근성을 준수하는 HTML 구조설계로 퍼블리싱을 하고 있습니다. 웹표준을 위해 HTML5에서 권장하는 마크업방식과 시멘틱 태그를 이용하여 전체 레이아웃을 설계합니다.

CSS

CSS 코딩은 크로스브라우징을 위해 안정적인 CSS 속성을 사용하여 HTML 엘리먼트들을 배치하고 디자인합니다. 추가적으로 SCSS도 사용하여 중첩, 변수, 선언, 연산자들을 활용할 수 있습니다.

javascript & jquery

네비게이션, 탭메뉴, 아코디언 등 퍼블리싱에 필요한 상호작용을 위해서 플러그인을 따로 사용하지 않고, 순수한 스크립트를 사용하여 제작할 수 있습니다. 또한, 다양한 JS프레임워크 학습을 위하여 스크립트 공부를 계속 하고 있습니다.

responsive web

반응형 웹 제작에 있어서 크로스브라우징을 위해 안정적인 기초방식인 float, position, media query를 활용한 반응형 웹 제작경험을 충분히 가지고 있습니다. 또한 flex, grid 방식으로 반응형 웹 사이트 제작도 충분히 경험하였습니다.

mobile web

모바일 웹앱 제작 경험이 있습니다. UI/UX 측면에서 미적인 디자인뿐만 아니라 모바일 사용자 경험에 최적화를 위해 효율적인 컨텐츠 배치를 우선적으로 생각합니다.

photoshop

기본적인 포토샵을 하기 위해서 GTQ 1급 자격증 시험을 본 상태이며 (결과 대기 중입니다.), 퍼블리싱 이외에 디자인적인 부분도 필요하다고 생각합니다. 기본적인 기술은 다룰 줄 알고 있고, 실력 향상을 위해서 공부 중입니다.
